# Data Labelling Lab

## What I did
Used Label Studio to annotate a custom movie review dataset for binary sentiment classification (Positive/Negative).

## My Modification
Created a custom movie reviews dataset with 20 samples instead of the default dataset.

## Dataset
- File: movie_reviews.csv
- 20 movie reviews
- Labels: Positive / Negative

## Tools Used
- Label Studio

## Files
- movie_reviews.csv — custom input dataset
- annotations.json — exported labels from Label Studio
